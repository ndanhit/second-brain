# Knowledge Base Changelog

Rolling log of KB additions and updates. Newest entries on top.

Use `python3 knowledge-base/scripts/append_changelog.py` to add entries:

```bash
python3 knowledge-base/scripts/append_changelog.py \
  --summary "Onboarding sweep" \
  --added "2-areas/systems/notification-service.md: initial profile" \
  --updated "3-resources/concepts/dead-letter-queue.md: added retry semantics"
```
